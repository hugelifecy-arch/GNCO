# Contributing

## Setup

1. Use Node 20.
2. Install dependencies:
   ```bash
   cd global-fund-architect
   npm ci
   ```

## Before opening a PR

Run:

```bash
npm run lint
npm run typecheck
npm test
npm run build:compliance
```

## Compliance policy

Do not add language implying offers, guaranteed returns, fees/promotions, or investor solicitation.
All jurisdiction references must include primary-source citations.
